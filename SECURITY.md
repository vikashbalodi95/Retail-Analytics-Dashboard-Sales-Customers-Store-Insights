# Security Policy
Thank you for taking the time to help improve the security of this project.
Your efforts ensure a safer and more reliable experience for all users.
## Supported Versions
Security updates are provided for the latest version of this project.
Please always use the most recent release to ensure maximum security and stability.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older  | :x:                |

# What Happens After Reporting?
1. Your report will be acknowledged within 48 hours

2. The issue will be reviewed and verified

3. Fixes or patches will be prioritized

4. You will be notified once the vulnerability is resolved

5. If needed, a security release will be published.
